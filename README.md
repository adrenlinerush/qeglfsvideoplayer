# qeglfsvideoplayer

A very simple video player that uses QT5 to play videos using DRI/KMS without X or Wayland.

## Usage

<pre>
qeglfsvideoplayer videofile
</pre>

* q - quit the application
* p - toggle play/pause
* right arrow - fast forward
* left arrow - rewind
* up arrow - volume up
* down arrow - volume down
